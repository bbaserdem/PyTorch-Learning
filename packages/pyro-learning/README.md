# Pyro Learning

A workspace for learning Pyro probabilistic programming.

## Installation

This package is part of the ml-learning workspace. From the root directory:

```bash
uv sync
```

## CLI Usage

After installation, you can use the CLI:

```bash
# Show version
pyro-learn version

# Show environment info
pyro-learn info

# Say hello
pyro-learn hello YourName
```

## Practice Scripts

Practice scripts will be located in `scripts/pyro/`.
